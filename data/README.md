# 竞品项目数据

本目录存储数字能源行业竞品调查数据。

## 目录结构

```
data/
└── competitor-projects/
    └── {公司名}/
        └── {年份}.csv
```

## 数据字段

| 字段 | 说明 |
|------|------|
| 项目名称 | 项目全称 |
| 开发商 | 项目投资方/业主 |
| EPC | 工程总承包方 |
| 项目所在国家 | 项目地理位置 |
| 规模 | 项目容量 |
| 产品类型 | 技术类型 |
| 中标时间 | 中标公告日期 |
| 信息来源 | 原始链接 |
